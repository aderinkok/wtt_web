CSS Eklemenin 3 yolu

  • 3 farklı HTML sayfası oluşturun.(internal,external ve inline isimlerinde)
  • Her bir sayfaya bu sayfada bağlantı veriniz.
  • inline.html = satır içi
  • internal.html = dahili
  • external.html = harici
  • Satır içi dosyasında ilgili elemente satır içi css ekleme yapısı kullanarak ilgili değişikliği yapın.
  • Dahili dosyasında ilgili elemente dahili yapı kullanarak css ekleme yapısı kullanarak ilgili değişikliği yapın.
  • Harici dosyasında ilgili elemente harici dosya çağırma işlemi gerçekleştirerek css ekleme yapısı kullanarak ilgili değişikliği yapın.
